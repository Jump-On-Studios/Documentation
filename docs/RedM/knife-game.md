## 1. Installation
jo_fivefingerfillet is a standalone script. So you don't need a specific framework to use it. It's also mean it works with all framework (VORP, RedEM:RP, QBCore, â€¦).

- Drag and drop the resource in your resources folder
  - jo_fivefingerfillet
- Add this ensure in your server.cfg
  - `ensure jo_fivefingerfillet`
- Congratulation, the Knife game, aka Five finger filler script is ready to be use !

## 2. Usage
Go near a table and enter the command `/fff` to start the game.<br>
By default, all table are not available but you can add them in the config file.
