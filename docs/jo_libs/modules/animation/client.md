# Animation

Animation is a powerfull module to play animations in your script.

## JO Variables

### jo.animation.easeIn
Type : *float*:  
> A variable to set the ease **in** speed of animations  
> Default: 4.0

### jo.animation.easeOut
Type : *float*  
> A variable to set the ease **out** speed of animations  
> Default: -4.0
