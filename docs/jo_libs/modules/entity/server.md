# Entity

A library with usefull function to manage entity

## jo.entity.delete()
A function to delete an entity only if exist
### Syntax
```lua
jo.entity.deletey(entity)
```
#### Parameters
`entity` : *integer*
> The entity ID

### Example
```lua
local entity = 23494
jo.entity.delete(entity)
```
